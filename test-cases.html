SUITE 1 - PLAYLIST STATUS RECOGNITION

Importance

High

Pre-conditions
  • browser installed
  • example playlist is public
  • Spotify API is up and running
Test input

playlist link: https://open.spotify.com/playlist/28Ffkndp6PP79X5LgFTXcZ

Test steps
  1. Go to the app page.
  2. Enter playlist link to the "Enter public playlist link or ID" field.
  3. Click "Send" button.
  4. Verify the message showing on the screen.
Expected results
  1. "Send" button is clickable.
  2. The app is showing the message: PLEASE, FIND BELOW DETAILS OF THE REQUESTED PLAYLIST.

ZESTAW 1 - ROZPOZNAWANIE STATUSU PLAYLISTY

Importance

Wysoka

Pre-conditions
  • zainstalowana przeglądarka
  • przykładowa playlista jest publiczna
  • Spotify API działa
Test input

link do playlisty: https://open.spotify.com/playlist/28Ffkndp6PP79X5LgFTXcZ

Test steps
  1. Go to the app page.
  2. Wprowadzić link do playlist w pole "Enter public playlist link or ID".
  3. Kliknąć przycisk "Send".
  4. Zweryfikować treść wiadomości na ekranie.
Oczekiwany wynik
  1. Przycisk "Send" jest klikalny.
  2. Pojawia się komunikat: PLEASE, FIND BELOW DETAILS OF THE REQUESTED PLAYLIST.
